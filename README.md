# 👁️ Assistive Vision for Blind

## 🔍 Overview

This project is a computer vision-based assistive system developed using Google Colab. It detects objects in real-time and provides audio feedback to assist visually impaired individuals.

---

## 🎯 Problem Statement

Visually impaired individuals face challenges in identifying objects in their surroundings. This project aims to provide an AI-based assistive solution.

---

## 💡 Solution

Using a pre-trained object detection model (YOLO/OpenCV), the system detects objects from images/video and converts detected labels into speech output.

---

## 🚀 Features

* 📷 Object detection using pretrained model
* 🔊 Audio feedback (text-to-speech)
* 🧠 Computer vision-based recognition
* 📊 Works on images/video

---

## 🛠️ Tech Stack

* Python
* OpenCV
* YOLO
* Google Colab
* pyttsx3 / gTTS

---

## ▶️ How to Run

### Option 1: Run in Colab

Upload the `.ipynb` file to Google Colab and run all cells.

### Option 2: Run locally

```bash
pip install -r requirements.txt
python main.py
```
